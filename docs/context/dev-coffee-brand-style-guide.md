# Dev Coffee Brand Style Guide

**Status:** Complete
**Created:** 2026-01-16

---

## Brand Personality

**Core Identity:** "The Sophisticated Hacker"

Dev Coffee embodies the aesthetic of a skilled developer's environment—immersive, focused, and unapologetically technical—while maintaining polish and professionalism. It's the visual language of late-night coding sessions, terminal windows, and hardware tinkering, elevated to a refined brand experience.

---

## Visual Mood

**Primary Atmosphere:** Dark, Immersive, Technical

The brand lives in darkness. Not empty or cold, but the purposeful darkness of a developer's den—monitors glowing in a dim room, the hum of hardware, the focus of deep work. This creates an intimate, professional environment that feels both serious and inviting to technical audiences.

**Retro-Tech Fusion:** The design deliberately references vintage computing (CRT monitors, scanlines, technical grids) while using modern techniques. This creates nostalgia without feeling dated—honoring the craft's history while looking forward.

---

## Color Story

| Role                 | Color                     | Feeling                                                                                                       |
| -------------------- | ------------------------- | ------------------------------------------------------------------------------------------------------------- |
| **Primary Accent**   | Emerald Green (`#10b981`) | The "terminal green" of classic computing. Signals life, activity, success. Used sparingly for maximum impact |
| **Background**       | Near-Black (`#030303`)    | Deep, absorbing void that lets content breathe                                                                |
| **Text**             | Soft White (90% opacity)  | Never harsh pure white—always slightly muted for comfort                                                      |
| **Status: WIP**      | Amber (`#f59e0b`)         | Work-in-progress, development                                                                                 |
| **Status: Stable**   | Blue (`#3b82f6`)          | Stable, production-ready                                                                                      |
| **Status: Research** | Purple (`#a855f7`)        | Experimental, ML/research                                                                                     |

**Philosophy:** Color is earned. The dark foundation makes every accent meaningful. Green isn't decoration—it's a signal of importance.

---

## Typography Philosophy

- **Monospace First** — Primary text uses coding fonts (JetBrains Mono). Reinforces developer identity
- **Clean Sans for Body** — Longer content uses modern, readable sans-serif (Geist). Technical doesn't mean hard to read
- **Uppercase Labels** — Section headers and UI labels are uppercase, widely spaced—like system diagnostics

## Tone & Voice

| Attribute     | Description                                    |
| ------------- | ---------------------------------------------- |
| **Confident** | Knows its craft. Doesn't oversell              |
| **Technical** | Embraces complexity. Doesn't dumb down         |
| **Focused**   | Direct communication. No fluff                 |
| **Warm**      | Professional, not cold. Approachable expertise |

**Language Style:** Direct, technical vocabulary is welcome. Avoid marketing-speak. Speak to peers, not prospects.

---

## Application Guidelines

### Do

- Start with darkness—let the void be the default
- Use emerald green only for the most important elements
- Prefer monospace fonts for headings and labels
- Include subtle technical textures (grids, scanlines) at very low opacity
- Cards and panels should feel like floating terminal windows
- Animation should be subtle, smooth—never flashy

### Avoid

- Bright or colorful backgrounds
- Rounded, playful shapes
- Casual or trendy typography
- Gradients as decoration
- Stock photography or generic tech imagery

## Emotional Summary

Dev Coffee should feel like:

- Walking into a well-equipped workshop at midnight
- The satisfying hum of a powerful system under load
- Respect between craftspeople who know their tools
- Focus, competence, and quiet confidence
